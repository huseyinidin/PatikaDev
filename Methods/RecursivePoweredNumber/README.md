# Ödev - Üs Hesabý Yapan Program

Recursive Metotlar ile Üslü Sayý Hesaplama

Java dilinde, taban ve üs deðerleri kullanýcýdan alýnan üs alma iþlemini **"Recursive"** metot kullanarak yapan programý yazýnýz.

Senaryo

```
Taban deðeri giriniz :2
Üs deðerini giriniz : 0
Sonuç : 1
Taban deðeri giriniz : 2
Üs deðerini giriniz : 3
Sonuç : 8
Taban deðeri giriniz : 5
Üs deðerini giriniz : 3
Sonuç : 125
```
