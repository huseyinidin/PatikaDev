# Ödev - Desene Göre Metot Oluþturma
Java dilinde kullanýcýdan alýnan n deðerine göre aþaðýdaki kurala uyan döngü kullanmadan bir **"Recursive"** metot yazýn.

**Kural:** Girilen sayý 0 veya negatif olduðu yere kadar girilen sayýdan 5 rakamýný çýkarýn. Her çýkarma iþlemi sýrasýnda ekrana son deðeri yazdýrýn. Sayý negatif veya 0 olduktan sonra tekrar 5 ekleyin. Yine her ekleme iþleminde sayýnýn son deðerini ekrana yazdýrýn.


## Senaryolar

```
N Sayýsý : 16
Çýktýsý : 16 11 6 1 -4 1 6 11 16 
```
```
N Sayýsý : 10
Çýktýsý : 10 5 0 5 10 
```
```
N Sayýsý : 25
Çýktýsý : 25 20 15 10 5 0 5 10 15 20 25 
```
```
N Sayýsý : 5
Çýktýsý : 5 0 5 
```
