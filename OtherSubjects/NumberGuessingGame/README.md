# Sayý Tahmin Oyunu

* 0-100 Arasýnda rastgele sayý üretilecek.

* 0-100 Arasýnda sayý olmalýdýr.

* Kullanýcýdan sayý girilmesi istenir ve rastgele üretilen sayýnýn altýnda ya da üzerinde olduðu söylenir. 

* Kullanýcýnýn en fazla 5 hakký vardýr. 

* Girilen sayý rastgele üretilen sayýya eþit ise kazandý deðilse kaybedene kadar haklarýný kullanabilir.
