# Ödev

**Dizideki Elemanlarý Sýralama**

Java dilinde, dizideki elemanlarý küçükten büyüðe sýralayan programý yazýnýz. Dizinin boyutunu ve dizinin elemanlarýný kullanýcýdan alýnýz.

```
Dizinin boyutu n : 5
Dizinin elemanlarýný giriniz :
1. Elemaný : 99
2. Elemaný : -2
3. Elemaný : -2121
4. Elemaný : 1
5. Elemaný : 0
Sýralama : -2121 -2 0 1 99
```

```
Dizinin boyutu n : 6
Dizinin elemanlarýný giriniz :
1. Elemaný : 1000221
2. Elemaný : 22
3. Elemaný : -1
4. Elemaný : 999
5. Elemaný : 0
6. Elemaný : 254
Sýralama : -1 0 22 254 999 1000221 
```
