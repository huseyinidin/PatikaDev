# Ödev 

Java dilinde, çok boyutlu diziler ile oluþturulmuþ matrisin transpozunu bulan programý yazýnýz.

**Matrisin transpozunu (devriðini) almak, matrisin ayný numaralý satýrlarý ile sütunlarýnýn yer deðiþtirmesi demektir.** Bu iþlem sonucu elde edilen matris, baþlangýçtaki matrisin transpozudur (devriðidir). Bu aþamada kxn’lik bir matrisin transpozu (devriði) nxk’lik bir matris olur. Örneðin bir A matrisimiz olsun. Bu A matrisinin transpozu (devriði), A^T (A üzeri T) ile gösterilir.

Örneðin aþaðýdaki 2x3’lik A matrisinin transpozu (devriði), 2x3’lük bir A^T matrisidir.

## Senaryo
```
Matris : 
1    2    3    
4    5    6    
Transpoze : 
1    4    
2    5    
3    6  
```
